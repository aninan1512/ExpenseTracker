# Expense Tracker (Java)

A simple Java console application to track daily expenses and store them persistently in a CSV file.

## Features
- Add categorized expenses
- Save to `expenses.csv` automatically
- View all saved expenses
- Calculate total spending

## Tech Stack
- Java (JDK 21)
- IntelliJ IDEA
- File I/O (CSV)

## How to Run
1. Clone the repo
2. Open in IntelliJ IDEA
3. Ensure Project SDK is set to JDK 21
4. Run `ExpenseTracker.java`

## Future Improvements
